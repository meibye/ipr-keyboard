# scripts/extras/

BLE support tools installed by `scripts/ble/ble_setup_extras.sh`.

## Files

| File | Installed Path | Purpose |
|---|---|---|
| `ipr_ble_diagnostics.sh` | `/usr/local/bin/ipr_ble_diagnostics.sh` | BLE health checks (adapter, services, UUID visibility, FIFO signals) |
| `ipr_ble_hid_analyzer.py` | `/usr/local/bin/ipr_ble_hid_analyzer.py` | BLE HID report inspection tool |

## Install Command

```bash
sudo ./scripts/ble/ble_setup_extras.sh
```
