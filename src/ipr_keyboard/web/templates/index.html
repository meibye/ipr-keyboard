IPR Keyboard Web Server

Welcome! This server provides the following endpoints and features:

  • /health – Health check (returns status: ok)
  • /status – System status (env, config, logs, BLE services, Bluetooth info)
  • /config/* – Configuration management (API)
  • /logs/* – Log access (API)
  • /pairing/* – Bluetooth pairing wizard and API

For a human-readable system status, visit /status.
